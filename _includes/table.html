The table

A tapestry to weave a narrative around.

When it comes to simplicity the table still remains supreme. Any dislike for this method of displaying tabular data can only be due to crimes committed against it by people using these for data exploration rather than sharing a distinct point. While I absolutely love LaTeX and markdown tables, straight to the point methods with no distractions, I've also started to become partial to gtables. I feel gtables are able to quickl y point a user to your argument a few pages deep into a report.
